This is a simple DBMS project on a Payroll Management System where employees can check their profile, attendance, leaves taken, salary, deductions, etc.

Framework used : Django.

Database has been implemented using sqlite3.

FrontEnd has been developed using html, css and bootstrap.

The “Payroll Management System” is a web based application for computerized management
of the payroll of the employees in a company. This application allows employees to login and
view their profile, details of attendance, leave, salary and deductions, and also helps the
company to keep track and manage all the payroll related details of its employees in an
organised and systematic manner, thus helping the productivity of the company at large.
